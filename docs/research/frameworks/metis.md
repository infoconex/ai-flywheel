# Metis Research Dossier

> **Research status:** Working dossier. Findings should be validated against primary sources before being treated as final comparative conclusions.

## Research Objective

Determine how closely Metis approaches the AI Flywheel concept of moving responsibility between procedural or textual knowledge and executable code.

## Working Hypothesis

Metis is expected to be one of the closest comparisons because it appears to distinguish textual memory from executable code memory and to promote reusable behavior into callable tools.

The central comparison is whether that representation choice is equivalent to the AI Flywheel's broader three-layer responsibility model and whether failure evidence explicitly routes changes among code, procedure, and runtime judgment.

## Evidence to Collect

- Formal definitions of text memory and code memory.
- Promotion or selection criteria between representations.
- Whether movement can occur in both directions.
- Whether procedural control is distinct from general textual memory.
- How failures influence representation changes.
- Whether responsibility allocation itself is treated as an optimization target.

## Provisional Position

Assume substantial overlap with the moving-boundary idea until rigorous comparison demonstrates otherwise. This dossier is critical to any eventual differentiation claim.
