# Framework Research Dossiers

This section examines related agent frameworks and methodologies individually. Each dossier summarizes the framework's relevant concepts, maps them to the AI Flywheel principles, and records areas of overlap and potential differentiation for further validation.

## Table of Contents

- [GenericAgent](genericagent.md) — Examines autonomous execution, skill crystallization, reusable SOPs, and executable capabilities.
- [Metis](metis.md) — Examines hybrid textual and executable memory, including the movement of stable recurring behavior into callable code.
- [MOSS](moss.md) — Examines source-level self-modification and evidence-driven evolution of agent software.
- [MASFly](masfly.md) — Examines SOP-driven multi-agent system construction, execution monitoring, and accumulated collaboration experience.
- [Ralph](ralph.md) — Examines iterative autonomous agent loops that repeatedly act against a persistent goal or specification.
- [Reflexion](reflexion.md) — Examines feedback-driven reflection and persistent linguistic memory used to improve future attempts.

For the principle-oriented view, see the [principle research dossiers](../principles/README.md). For a condensed cross-framework view, see the [framework comparison matrix](../framework-comparison-matrix.md).
